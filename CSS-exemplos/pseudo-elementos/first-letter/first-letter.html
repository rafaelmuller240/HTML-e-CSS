Você pode usar o pseudoelemento ::first-letter para adicionar um efeito especial ao primeiro caractere de um texto!

Nota: O pseudoelemento ::first-letter só pode ser aplicado a elementos de nível de bloco.

As seguintes propriedades se aplicam ao pseudoelemento ::first-letter:

  • font properties
  • color properties
  • margin properties/li>
  • padding properties
  • border properties
  • text-decoration
  • vertical-align (somente se "float" for "none")
  • text-transform
  • line-height
  • float
  • clear
